# Simply VR Game "Toxic Frogs"

<img src="Document%20Images/Frog%20Image.png" width="50%" height="70%">


### Description
As standard for VR games, Player character control have first-person perspective.
- Basic FPC (First Person Controller):
    - Continuous Move Provider
    - Continuous Turn Provider
    - Character Controll Driver - controls the change of the controller height

Also important is  Hand Grabbing mechanism that allow grab physical items by Player. An important element of the gameplay is the 'Shot mini game'. 
- Main items of mini Shootgame:
    - Generator
    - Scenario
    - Mission Panel
    - Trophies

| Name             | Role |
| :---------------- | :------: |
| Generator        |   Main Mechanism, create Frogs and Trigger endGame   |
| Scenario           |   Set parameters for current level   |
| Mission Panel    |  Show information, ensure mission activation   |
| Trophies |  Unlocks after completing the mission   |
